Tag Example Description

<select> Dropdown menu for form input.
<textarea> Multiline text input for forms.

Common HTML Setup

Example Description 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Webpage</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>

  <h1>Hello World!</h1>

</body>
</html>
 HTML5 page boilerplate with meta viewport and stylesheet link.
<link rel="stylesheet" href="style.css"> Links an external CSS stylesheet to your HTML file.
<script src="script.js"></script> Links an external JavaScript file.
<img src="images/photo.jpg" alt="Description of image"> Embeds an image with required alt text for accessibility.
<a href="https://www.example.com">Click Here</a> Creates a clickable hyperlink.
<!-- This is a comment --> Inserts a comment that will not be displayed on the page. 
<table>
  <tr>
    <th>Heading 1</th>
    <th>Heading 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>
 Defines a basic HTML table with headings and rows. 
<form action="/submit" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>
 Basic form template with label, text input, and submit button.
